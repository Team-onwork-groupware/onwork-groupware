# UC-ONB-0529: 온보딩 튜토리얼 API/UI 정합화

## Goal

명세의 온보딩 튜토리얼 API를 구현하고, 첫 로그인 또는 홈 진입 시 필요한 안내를 자동 노출한다.

## Actors

- 신규 직원: 첫 로그인 후 시스템 사용법을 단계적으로 확인한다.
- 기존 직원: 필요할 때 온보딩을 재시작한다.
- 시스템: 사용자별 튜토리얼 진행 상태와 표시 여부를 저장한다.

## Main Flow

1. 사용자가 홈에 진입하면 `GET /api/v1/onboarding/tutorials/me`로 노출할 튜토리얼을 조회한다.
2. 시스템은 사용자 역할과 완료 여부에 맞는 튜토리얼을 반환한다.
3. 사용자가 다음 단계를 진행하면 step API로 현재 단계를 저장한다.
4. 사용자가 나중에 보기 또는 다시 보지 않기를 선택하면 visibility API로 상태를 저장한다.
5. 사용자가 설정 또는 도움말에서 restart API로 튜토리얼을 재시작한다.

## Rules

- 튜토리얼 상태는 사용자별로 저장한다.
- 홈 자동 노출은 업무 화면을 막지 않도록 닫기/나중에 옵션을 제공한다.
- 역할별로 불필요한 단계는 노출하지 않는다.
- 응답 필드는 `snake_case`를 사용한다.

## Acceptance Criteria

- `/onboarding/tutorials/me`, step, visibility, restart API가 동작한다.
- 신규 직원 첫 로그인 경로에서 튜토리얼이 표시된다.
- 사용자가 숨김 처리한 튜토리얼은 재시작 전까지 자동 노출되지 않는다.
- 재시작 후 첫 단계부터 다시 진행할 수 있다.
