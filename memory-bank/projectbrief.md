# Project Brief

## Project Name

OnWork

## Purpose

AI 에이전트를 활용해 개발 과정의 반복 작업을 표준화하고 자동화한다.

## Core Problems

- LLM이 긴 개발 과정에서 계획을 잊는다.
- 테스트 작성/실행이 누락된다.
- 문서화가 누락된다.
- 커밋 히스토리와 PR 설명이 부실하다.
- 사람이 매번 같은 검증 절차를 반복해야 한다.

## Target Outcome

- 계획/구현/테스트/문서화/커밋 workflow
- 역할별 에이전트 템플릿
- CI 기반 검증 구조
